# RTCrypt
RTCrypt es la capa de seguridad de RT Stack: define perfiles y herramientas para integrar cifrado, integridad, autenticación y control de acceso usando estándares abiertos como TLS, COSE, OIDC y KMS. Su objetivo es proteger RTPack, RTCore, RTStream y RTCloud sin inventar criptografía casera ni romper interoperabilidad. En entornos reales y seguros.
