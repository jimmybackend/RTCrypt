# Scripts

Scripts auxiliares del repositorio.

## Posibles utilidades futuras

- generación de fixtures;
- validación rápida de esquemas;
- empaquetado de test vectors;
- verificación de consistencia entre perfiles y ejemplos.

## Regla

Los scripts no deben convertirse en lógica central del proyecto.
La lógica central debe vivir en los módulos de implementación o en herramientas reproducibles del workspace.
